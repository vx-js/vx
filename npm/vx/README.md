# @vx-js/vx

`vx` is a fast, cache-first npm package installer prototype written in Rust.

## Install

```bash
npm i -g @vx-js/vx
```

Supported platforms (via optional dependencies):

- Windows x64 (MSVC)
- Linux x64 (GNU)
- Linux arm64 (GNU)
- macOS x64
- macOS arm64

## Usage

```bash
vx --help
```
